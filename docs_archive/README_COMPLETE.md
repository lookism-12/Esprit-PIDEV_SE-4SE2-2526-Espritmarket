## 🎓 Pour les Développeurs

### Setup
```bash
# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
npm install
npm start
```

### URLs
- Frontend: http://localhost:4200
- Backend: http://localhost:8090
- Seller Marketplace: http://localhost:4200/seller/marketplace
